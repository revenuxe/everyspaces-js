# EverySpaces Interiors

Marketing and app UI for EverySpaces, built with **Next.js** (App Router), React, TypeScript, Tailwind CSS, and shadcn/ui.

## Local development

Requires [Node.js](https://nodejs.org/) and npm.

```sh
npm install
npm run dev
```

Dev server: [http://localhost:8080](http://localhost:8080)

## Scripts

| Command            | Description              |
| ------------------ | ------------------------ |
| `npm run dev`      | Next.js dev (port 8080)  |
| `npm run build`    | Production build         |
| `npm run start`    | Start production server  |
| `npm run lint`     | ESLint                   |
| `npm run typecheck`| TypeScript (no emit)     |

## Stack

- Next.js, React, TypeScript
- Tailwind CSS, shadcn/ui
- Supabase, TanStack Query

Deploy on [Vercel](https://vercel.com/) or any Node host that runs `next build` / `next start`.
